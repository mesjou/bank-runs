# Bank Runs

Multi-Agent Environment that recreates Bank Run experiments conducted by Arifovic (2019). 
Agents based on Q-learning learn similar strategies as human participants based on incentives to cooperate or defect.
For more information about the Diamond-like bank run modeling see:
[Arifovic, Jasmina. "Evolution of sunspot like behavior in the agent based economies of bank runs." Journal of Evolutionary Economics 29.1 (2019): 365-389.](http://www.sfu.ca/~arifovic/Papers/IELsunspots.pdf)
